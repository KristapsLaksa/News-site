# News-site

This project  aggregates top news headlines in USA splitting them by categories.


## Authors

- [@KristapsLaksa](https://github.com/KristapsLaksa)



## API Reference


```http
https://newsapi.org/
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `NEWS_API_KEY` | `string` | **Required**. Your API key |
